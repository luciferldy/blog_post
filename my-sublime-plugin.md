Sublime Text 3 的插件是编码的利器，这里我介绍了自己安装的一些插件。

* Alignment  代码对齐（基本没用）
* BracketHighlighter  高亮功能（基本没用 too）
* DocBlockr  php 代码检查
* Emmet  html 代码补全，前端开发很方便
* JsFormat  JS 代码补全
* Markdown Preview	markdown 语法检查和效果预览
* SideBar	增加右键功能
* CodeFormatter  格式化代码
* WakaTime 统计在编辑器上写代码的时间

后面如果有插件更新也会同步到博客上。
